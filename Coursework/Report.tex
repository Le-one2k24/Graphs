	\section*{Введение}
	\addcontentsline{toc}{section}{Введение}
	
	Целью курсовой работы является разработка экспертной системы, основанной на продукционной модели представления знаний, для определения типа аллергии по симптомам пользователя. Система реализована в среде CLIPS и использует бинарное дерево решений для интерактивного опроса пользователя и вывода наиболее вероятного типа аллергии.
	
	\section{Общие сведения}
	Аллергия — патологическая реакция иммунной системы на внешние или внутренние раздражители (аллергены). Тип аллергии определяется по совокупности симптомов: кожные проявления, респираторные симптомы, желудочно-кишечные реакции, связь с контактами и сезонностью.
	
	\subsection{Перечень диагностируемых типов}
	Перечень типов аллергических реакций включает следующие варианты:
	\begin{itemize}
		\item \textbf{Аллергия на яд перепончатокрылых (ос, пчёл)}
		\item \textbf{Анафилаксия на лекарственные средства}
		\item \textbf{Пищевая анафилаксия (тяжёлая IgE-опосредованная)}
		\item \textbf{Анафилаксия неясной этиологии}
		\item \textbf{Сезонный аллергический ринит (поллиноз)}
		\item \textbf{Круглогодичный аллергический ринит (на бытовые аллергены: клещи, плесень)}
		\item \textbf{Аллергическая бронхиальная астма}
		\item \textbf{Неаллергический ринит}
		\item \textbf{Контактная крапивница (IgE-опосредованная)}
		\item \textbf{Аллергический контактный дерматит (IV тип, замедленный)}
		\item \textbf{Хроническая спонтанная крапивница}
		\item \textbf{Атопический дерматит (экзема)}
		\item \textbf{Пищевая аллергия (молоко/яйцо у детей до 2 лет)}
		\item \textbf{Пищевая аллергия (злаки/рыба и др.)}
		\item \textbf{Синдром оральной аллергии (на свежие фрукты/овощи)}
		\item \textbf{Холинергическая крапивница / анафилаксия, связанная с физической нагрузкой}
		\item \textbf{Крапивница от физической нагрузки (не связана с едой)}
		\item \textbf{Аллергический контактный дерматит (замедленная реакция при локальном контакте)}
		\item \textbf{Лекарственная сыпь (IV тип, замедленная)}
		\item \textbf{Системная реакция замедленного типа на укус насекомого}
		\item \textbf{Физическая крапивница (холодовая/солнечная/вибрационная)}
		\item \textbf{Идиопатическая крапивница}
		\item \textbf{Неопределённый тип (требуется обследование у аллерголога)}
	\end{itemize}

	\section{Требования к системе}
	
	\subsection{Функциональные требования}
	\begin{enumerate}
		\item Интерактивный опрос пользователя по набору симптомов и условий.
		\item Переход по бинарному дереву решений на основе ответов.
		\item Применение продукционных правил для вывода промежуточных фактов и финального диагноза.
		\item Вывод диагноза и краткого объяснения, какие симптомы и правила привели к выводу.
	\end{enumerate}
	
	\subsection{Нефункциональные требования}
	\begin{itemize}
		\item Реализация в среде CLIPS (версия 6.30 или совместимая).
		\item Использование продукционной модели представления знаний.
		\item Наличие бинарного дерева решений с не менее чем четырьмя ярусами и тридцатью узлами.
		\item Возможность расширения базы знаний без изменения механизма вывода.
	\end{itemize}
	
	\section{Модель представления знаний}
	
	\subsection{Продукционная модель}
	В системе используется продукционная модель: факты в рабочей памяти и правила вида \texttt{IF ... THEN ...}. Правила формируют промежуточные факты и финальный факт диагноза. Механизм вывода — прямой (forward chaining), реализуемый средствами CLIPS.
	
	\subsection{Шаблоны фактов}
	
	В данной экспертной системе не используются структурированные шаблоны
	(\texttt{deftemplate}). Вместо них применяются простые маркерные факты,
	которые управляют переходами по бинарному дереву решений и фиксируют
	достижение конечного диагноза.
	
	Ниже приведены реальные типы фактов, используемые системой:
	
	\begin{lstlisting}
		; Факт, указывающий текущий активный узел дерева
		(next-node q1)
		
		; Факт перехода к следующему узлу (создаётся функцией ask-question)
		(assert (next-node q2_yes))
		
		; Факт финального диагноза (выводится в листьях)
		; В коде диагноз печатается через printout, но логически он эквивалентен факту:
		(diagnosis "Аллергия на яд перепончатокрылых (ос, пчёл)")
		
		; Ответ пользователя хранится во временной переменной
		(bind ?answer (read))
	\end{lstlisting}
	
	\subsection{Рабочая память}
	
	Рабочая память содержит следующие виды данных:
	
	\begin{itemize}
		\item факты вида \texttt{(next-node ...)}, определяющие текущий узел дерева решений;
		\item факты, создаваемые функцией \texttt{ask-question}, которые направляют систему по ветвям дерева;
		\item временные значения ответов пользователя, полученные через \texttt{(read)};
		\item финальный вывод, формируемый в листьях дерева и выводимый через \texttt{printout}.
	\end{itemize}
	
	\section{Бинарное дерево решений}
	
	\subsection{Описание структуры дерева}
	В отличие от простого семантического дерева, используемая в системе структура — это диагностическое бинарное дерево, корнем которого является вопрос о временном профиле реакции: \emph{«Реакция возникает в течение 1–6 часов после контакта с аллергеном?»}. Это ключевой вопрос, который разделяет немедленные (включая анафилаксию и большинство IgE‑опосредованных реакций) и отсроченные/специфические реакции.
	
	Далее дерево разворачивается по следующим логическим ветвям (кратко):
	\begin{itemize}
		\item \textbf{Если да (немедленная реакция)}: уточняется наличие анафилаксии; при анафилаксии проверяются возможные триггеры (укусы перепончатокрылых, внутривенные лекарства, пищевые продукты) и формируются соответствующие листья (яд насекомых, лекарственная анафилаксия, пищевая анафилаксия, анафилаксия неясной этиологии). Если анафилаксия отсутствует, далее проверяется вовлечение дыхательной системы, кожа или ЖКТ, с ветвлением на поллиноз, круглогодичный ринит, астму, контактные реакции, пищевые реакции, физические триггеры и т.д.
		\item \textbf{Если нет (не немедленная)}: проверяется, возникает ли реакция через 12–24 часа и более; при положительном ответе — ветвление на замедленные кожные реакции (локальный контактный дерматит, лекарственная сыпь) и системные замедленные реакции после укуса насекомого. Если реакция не соответствует ни немедленному, ни типичному замедленному профилю, проверяются физические триггеры (холод/тепло/солнце) и возможные физические крапивницы или идиопатическая крапивница.
	\end{itemize}
	
	\subsection{Пояснение выбора корневого вопроса и порядка проверок}
	Корневой вопрос о временном профиле реакции обеспечивает быстрое разделение множества клинических сценариев: немедленные реакции требуют немедленного внимания и имеют характерные наборы симптомов и триггеров, тогда как отсроченные реакции чаще связаны с контактами, лекарствами или иммунными механизмами замедленного типа. Такой порядок уменьшает среднюю глубину поиска и делает дерево более информативным для врача‑эксперта и пользователя.

	\section{Реализация в CLIPS}
	
	\subsection{Общий подход}
	Интерфейс — консольный. Система задаёт вопросы, формирует факты и перемещается по узлам дерева с помощью факта \texttt{(next-node ...)}. Для удобства реализована функция-обёртка \texttt{ask-question}, которая запрашивает ответ \texttt{yes} или \texttt{no} и добавляет соответствующий факт в рабочую память. Правила соответствуют ветвям диагностического дерева.
	
	\subsection{Примеры правил, соответствующие реальному дереву (фрагменты)}
	
	Ниже приведены примеры продукционных правил, которые действительно
	используются в системе и соответствуют структуре дерева из файла
	\texttt{Tree.txt}. Каждое правило реагирует на факт \texttt{(next-node ...)}
	и либо задаёт следующий вопрос, либо завершает диагностику.
	
	\paragraph{Анафилаксия после укуса перепончатокрылых}
	
	Если пользователь находится в узле \texttt{q3\_anaph} и отвечает «да» на вопрос о том, что реакцию вызвало жалящее насекомое, система переходит в лист \texttt{leaf\_bee} и выводит диагноз.
	
	CLIPS‑фрагмент:
	\begin{lstlisting}
		(defrule q3_anaph
		(next-node q3_anaph)
		=>
		(ask-question 
		"Реакцию вызвало жалящее насекомое?"
		leaf_bee
		q4_anaph_no))
		
		(defrule leaf_bee
		(next-node leaf_bee)
		=>
		(printout t 
		"Диагноз: Аллергия на яд перепончатокрылых (ос, пчёл)" crlf)
		(printout t "--- Диагностика завершена ---" crlf)
		(halt))
	\end{lstlisting}
	
	\paragraph{Пищевая анафилаксия}
	
	Если анафилаксия не вызвана укусом насекомого и не связана с лекарствами, система уточняет, возникли ли симптомы в течение 30 минут после употребления орехов/моллюсков. При ответе «да» — диагноз пищевая анафилаксия.
	
	CLIPS‑фрагмент:
	\begin{lstlisting}
		(defrule q5_anaph_food
		(next-node q5_anaph_food)
		=>
		(ask-question 
		"Симптомы начались в течение 30 минут после употребления арахиса/орехов/моллюсков?"
		leaf_anaph_food
		leaf_anaph_unknown))
		
		(defrule leaf_anaph_food
		(next-node leaf_anaph_food)
		=>
		(printout t 
		"Диагноз: Анафилаксия пищевая (тяжёлая IgE-опосредованная)" crlf)
		(halt))
	\end{lstlisting}
	
	\paragraph{Сезонный аллергический ринит (поллиноз)}
	
	Если реакция связана с дыхательной системой и симптомы появляются сезонно во время цветения, система делает вывод о поллинозе.
	
	CLIPS‑фрагмент:
	\begin{lstlisting}
		(defrule q4_season
		(next-node q4_season)
		=>
		(ask-question 
		"Симптомы (чихание, зуд в носу, ринорея) появляются сезонно во время цветения?"
		leaf_seasonal
		q5_perennial))
		
		(defrule leaf_seasonal
		(next-node leaf_seasonal)
		=>
		(printout t 
		"Диагноз: Сезонный аллергический ринит (поллиноз)" crlf)
		(halt))
	\end{lstlisting}
	
	\paragraph{Аллергический контактный дерматит (замедленная реакция)}
	
	Если реакция возникает через 12–48 часов после контакта с металлом/косметикой и похожа на экзему, система делает вывод о контактном дерматите IV типа.
	
	CLIPS‑фрагмент:
	\begin{lstlisting}
		(defrule q6_delayed
		(next-node q6_delayed)
		=>
		(ask-question 
		"Сыпь появилась через 12–48 часов после контакта с металлом/косметикой, похожа на экзему?"
		leaf_delayed_contact
		q7_wheals))
		
		(defrule leaf_delayed_contact
		(next-node leaf_delayed_contact)
		=>
		(printout t 
		"Диагноз: Аллергический контактный дерматит (IV тип, замедленный)" crlf)
		(halt))
	\end{lstlisting}
	
	\subsection{Объяснение вывода}
	
	Каждый переход по дереву фиксируется фактом \texttt{(next-node ...)}.
	Таким образом система сохраняет путь, по которому пользователь прошёл
	в процессе диагностики. Листовые правила выводят диагноз и завершают
	работу, что позволяет однозначно восстановить последовательность
	сработавших правил и вопросов, приведших к результату.
	
	\section{Тестирование}
	
	\subsection{Сценарии тестирования}
	\begin{enumerate}
		\item \textbf{Анафилаксия после еды}: немедленная реакция, анафилаксия, связь с орехами $\rightarrow$ пищевая анафилаксия.
		\item \textbf{Сезонный ринит}: быстрая реакция, без анафилаксии, чихание, слезотечение, ухудшение весной $\rightarrow$ поллиноз.
		\item \textbf{Контактный дерматит}: немедленная реакция, без анафилаксии, локальная экземапосле контакта с металлом $\rightarrow$ контактный дерматит.
		\item \textbf{Физическая крапивница}: реакция от 6 до 12 часов, на холод/тепло/солнце появляются волдыри $\rightarrow$ физическая крапивница или холинергическая форма.
	\end{enumerate}
